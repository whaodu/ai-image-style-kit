# 🔥 豆包图像工具箱

火山引擎豆包 × OpenClaw 生图 Skill。让你用自然语言驱动 AI 生图，支持「风格提取 → 保存 → 复用」全流程。

---

## 安装步骤

### 第一步：安装 Skill

通过 ClawHub 安装（推荐）：

```bash
openclaw skill install ai-image-style-kit
```

或手动安装：将整个目录克隆到你的 skills 目录：

```bash
# 克隆到 OpenClaw skills 目录
git clone <repo_url> ~/.openclaw/skills/ai-image-style-kit
```

### 第二步：配置 API Key

Skill 需要调用火山引擎豆包 API，你需要准备：

1. **注册火山引擎账号** → https://console.volcengine.com/ark
2. **创建 ARK API Key** → 控制台 → API Key 管理 → 创建密钥
3. **在 OpenClaw 的 `TOOLS.md` 中配置**（或直接设置环境变量）：

```bash
# 在 TOOLS.md 或你的环境变量中添加：
export ARK_API_KEY="你的ARK-API-Key"
```

> API Key 只用于生图，不会保存到任何外部服务。

### 第三步：（推荐）配置飞书凭证

配置后，生成的图片可以直接发送到飞书对话，无需手动下载转发：

1. **创建飞书应用** → https://open.feishu.cn/app（本步骤可选）
2. **在 OpenClaw 的 `TOOLS.md` 中添加**（或直接设置环境变量）：

```bash
export FEISHU_APP_ID="cli_xxx"
export FEISHU_APP_SECRET="xxx"
```

> 如果不配置此步骤，图片生成功能不受影响，只是无法直接发送到飞书对话。

## 快速开始

安装完成后，直接用自然语言描述需求即可：

### ① 提取风格（一次性）

发一张喜欢的参考图，说「提取风格」：

```
你：发一张图 + "提取风格"
AI：✅ 风格提取完成，已保存（风格编号 #001）
```

### ② 用风格生成图片

说「用XX风格生成图片」：

```
你："用这个风格生成一张公众号封面图"
AI：已选中【知识卡片】风格，请描述具体内容？
你："AI出海赛道分析，科技感"
AI：生成中... → 图片直接发给你
```

## 参数说明

| 参数 | 说明 | 示例 |
|------|------|------|
| `--size=尺寸` | 生图尺寸 | `--size=16:9` / `--size=2k` / `--size=1920x1080` |
| `--no-watermark` | 关闭水印 | `--no-watermark` |
| `--name=名称` | 提取风格时命名 | `--name=科技感` |

**支持的比例格式：**

| 格式 | 输出分辨率 | 说明 |
|------|-----------|------|
| `16:9` | 2560×1440 | 默认，横版 |
| `4:3` | 2224×1664 | 横版经典比例 |
| `1:1` | 1920×1920 | 方版 |
| `9:16` | 1440×2560 | 竖版 |
| `1k/2k/3k/4k` | 多种 | 快模生成，快速出图 |

## 常见问题

**Q: 生成的图片比例不对？**

确保使用支持的尺寸格式，如 `--size=16:9`。不要用 `16:9` 以外的中文冒号写法。

**Q: 风格提取不准确？**

参考图尽量清晰、主体突出、有明确的视觉风格。复杂场景或多主体的图可能提取出混合风格。

**Q: 怎样查看已保存的风格？**

AI 会记住所有风格。你可以直接说「用上次那个手绘风格生成图片」，AI 自动匹配。

**Q: 如何删除某个风格？**

目前需要手动删除 `scripts/styles/style_XXX.json` 文件。
